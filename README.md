# Trekking Management Application

A web application for managing trekking activities, built for the Modern Application Development I (MAD-I) course project.

Admin, Trek Staff, and Users (Trekkers) can manage treks, staff approvals, and bookings through role-based dashboards.

## Frameworks

- Flask (backend)
- Jinja2, HTML, CSS, Bootstrap (frontend)
- SQLite (database)
